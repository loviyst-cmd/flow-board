# Flow Board · 极简看板

> 一个高颜值的 Trello 风格看板，纯单文件 HTML/CSS/JS 实现，零第三方依赖。
> 数据自动保存在浏览器本地，开箱即用。

![Flow Board 截图](https://via.placeholder.com/800x400/3b82f6/ffffff?text=Flow+Board+Kanban)

## ✨ 特性

- 🎨 **浅蓝渐变主题** - 清爽的冷色调界面，精心调校的视觉细节
- 📋 **多项目管理** - 支持创建多个独立看板，活跃/归档分类管理
- 🗂️ **完整 CRUD** - 列表和卡片的增删改查，全部支持
- 🎯 **卡片拖拽** - 跨列拖动卡片 + 列横向拖拽切换位置
- ✨ **FLIP 平滑动画** - 拖拽时周围卡片丝滑让位，绝不生硬闪现
- 📜 **边缘自动滚动** - 拖到看板边缘时自动横向滚动
- 🪟 **无缝弹窗** - 点击卡片从原位放大展开，关闭时缩回原位
- 🏷️ **多色标签** - 卡片支持自定义颜色标签
- ✅ **子任务清单** - 卡片内含完整子任务勾选清单
- 📅 **双日期** - 开始日期 + 截止日期，过期自动标红
- 🗑️ **回收站** - 删除的列表/卡片进回收站，可恢复
- 💾 **本地持久化** - 数据自动存浏览器 localStorage，刷新不丢
- 📤 **导入导出** - 支持 JSON 文件备份/恢复
- 📱 **响应式** - 适配桌面和移动端

## 🚀 快速开始

### 方式一：直接打开
下载 `flow-board.html`，双击用浏览器打开即可使用。

### 方式二：在线访问
部署到 Cloudflare Pages / GitHub Pages / Vercel 等任意静态托管。

## 🛠️ 技术栈

- 原生 HTML5 + CSS3 + JavaScript（ES6+）
- 零第三方库，零构建步骤
- CSS 自定义属性（变量）+ Flexbox + Grid 布局
- FLIP 动画技术实现拖拽让位
- Pointer Events API 实现统一拖拽
- localStorage 实现数据持久化

## 📖 使用说明

### 基本操作
- **新建项目**：左下角「+ 新建项目」
- **添加卡片**：列表底部「+ 添加卡片」
- **添加列表**：看板最右侧「+ 添加列表」
- **编辑卡片**：点击卡片标题即可编辑，或点击卡片打开详情弹窗
- **拖拽卡片**：按住卡片拖到其他列
- **拖拽列表**：按住列表标题拖到其他位置
- **删除**：卡片/列表右上角 🗑 按钮移到回收站

### 数据管理
- **自动保存**：所有操作即时保存到浏览器本地
- **导出备份**：右上角「导出」按钮，下载 JSON 文件
- **导入恢复**：右上角「导入」按钮，上传 JSON 文件
- **清空数据**：浏览器控制台执行 `clearAllData()`

## 🔧 自定义

修改 HTML 顶部的 CSS 变量即可换主题：

```css
:root{
  --bg-1:#eef4fb;      /* 背景渐变起点 */
  --bg-2:#dde9f5;      /* 背景渐变终点 */
  --accent:#3b82f6;    /* 主题强调色 */
  --card-bg:#ffffff;   /* 卡片背景 */
  /* ... 更多变量见文件顶部 */
}
```
